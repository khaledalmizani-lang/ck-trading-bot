# CK Trading Bot

## Setup on Render

### Environment Variables (required)
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID  
- TELEGRAM_CHANNEL_ID

### Start Command
```
python main.py
```

### Files needed
- main.py
- config.py
- fetcher.py
- strategy.py
- notifier.py
- commands.py
- journal.py
- analyzer.py
- report.py
- coingecko.py
